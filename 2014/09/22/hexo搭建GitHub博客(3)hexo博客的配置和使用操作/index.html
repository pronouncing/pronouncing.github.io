在上一节中，我们在本地和github搭建起了自己的hexo博客站点，但是还未发表过文章，站点的配置还是原来的默认值。在本节，我们来进行个性化的设置，将站点打造成自己的，同时介绍下怎么撰文和发表。
站点配置用到两个文件，一个是对整站的配置D:\hexo_config.yml，另一个是对主题的配置D:\hexo\themes_config.yml，我们来分别介绍。

先来介绍对整站配置： D:\hexo\ _config.yml
